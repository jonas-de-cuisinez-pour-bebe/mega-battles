# Mega Battles

Tactical-RPG de duel en 3D (Three.js) — reboot 2026 du game design de 2013.
Devise : « des duels simples et rapides ». Humans vs Zombies, post-apo cartoon.

## Lancer

```bash
npm install
npm run dev
```
